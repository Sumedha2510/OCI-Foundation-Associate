Question: You want to store the backup of a database in cloud storage for an extended period of time. Which type of storage should you configure for these files?
Answer: Archive Storage

Question: Which statement is true about the OCI Block storage service?
Answer: It can be attached to a compute instance

Question: Which statement is true about OCI File Storage?
Answer: It organizes files in a hierarchy of named directories

Question: Which storage option is NOT available in OCI?
Answer: NetApp Cloud Volume 

Question: You store multiple versions of objects in a bucket, but your IT team has asked you to delete any previous object versions 120 days after the object version transitions 
from the latest version to a previous version. Which OCI Object Storage feature can be used for this purpose?
Answer: Object Lifecycle Management
